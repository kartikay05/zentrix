# Zentrix Frontend

This is the frontend application for the **Zentrix AI Incident Response Platform**, built with React and Vite.

## 🚀 Features (Planned)
- **Real-time Incident Dashboard**: Monitor system health and AI alerts.
- **AI Response Console**: Interact with the AI engine to resolve incidents.
- **Workspace Management**: Configure organization settings and team roles.
- **Integration Hub**: Connect GitHub, Slack, and other dev tools.

## 🛠️ Development

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```

## 🏗️ Tech Stack
- **Framework**: [React](https://reactjs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: Vanilla CSS (Modern & Premium Aesthetics)
- **State Management**: Context API / Redux (Planned)
